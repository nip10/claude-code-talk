<TipSlide
  title={"Skills: Best Practices and Gotchas"}
  description={"Audit third-party skills before use and reserve skills for judgment-based tasks, not deterministic checks."}
  tags={["Config", "Workflow", "Gotcha"]}
>

- **Audit third-party skills:** Use open-source/community skills as starting points, but treat every imported skill as untrusted until reviewed. Check `skill.md` and referenced scripts before use.
- **Skills for judgment, hooks for checks:** Use skills for fuzzy, context-dependent workflows. Use hooks/scripts for deterministic tasks (format, lint, test, guardrails) that shouldn't depend on model judgment.

</TipSlide>
