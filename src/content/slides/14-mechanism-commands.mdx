<TipSlide
  title="Commands: Reusable On-Demand Workflows"
  description="Use slash commands when you want a repeatable operation you trigger manually."
  tags={["Workflow", "Commands"]}
>

Use when: a task repeats often and you want a fast, guided starting point.

Avoid when: you need automatic enforcement without user invocation.

```markdown
description: Create a concise implementation plan
argument-hint: [feature or ticket]

# Plan Feature

Read requirements for $ARGUMENTS,
propose a 3-step plan,
and include verification commands.
```

</TipSlide>
