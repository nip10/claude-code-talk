_Source section: Prompt Engineering_
- **What:** Generate structured documentation (PRD, architecture.md, decision.md, feature.json) and user stories covering all interaction paths before coding.
- **Why it matters:** Gives Claude the right context, dramatically reducing errors. User stories set a standard the implementation must meet, covering best-case and edge-case scenarios.
- **How:** Generate docs and user stories first. After implementation, have Claude work through them one by one as acceptance criteria.
