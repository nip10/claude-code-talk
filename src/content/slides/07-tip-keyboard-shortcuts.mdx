<TipSlide
  title="Keyboard Shortcuts"
  description="Speed up your workflow with these essential shortcuts."
  tags={["Workflow", "Productivity"]}
>

| Shortcut | Action |
|---|---|
| **Esc** | Interrupt current generation |
| **Double-tap Esc** (non-empty input) | Clear the input field |
| **Double-tap Esc** (empty input) | Rewind to a previous context checkpoint |
| **Ctrl+L** | Clear conversation |
| **Shift+Tab** | Toggle between Plan Mode and Accept Edits Mode |
| **Up Arrow** | Recall last prompt |
| **Ctrl+W** | View all sessions across the project |

</TipSlide>
