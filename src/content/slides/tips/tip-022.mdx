_Source section: Sub-Agents_
- **Use for isolated work:** Sub-agents have their own context windows. Only their summary returns. Create definitions in `.claude/agents/` with description, allowed tools, and task scope. Built-in agents: `explore`, `plan`, `general-purpose`.
- **Avoid for session-dependent tasks:** Sub-agents don't receive the parent's full reasoning trace. Keep testing, debugging, and code review in the main session — a testing agent that doesn't know what code was just written produces worse results.
