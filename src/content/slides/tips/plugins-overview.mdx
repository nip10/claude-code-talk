<TipSlide
  title={"Plugins: Superpowers, RALPH, and GSD"}
  description={"Three popular third-party plugins for structured planning, autonomous pipelines, and scope-discovery workflows."}
  tags={["Workflow", "Advanced"]}
>

- **Superpowers:** An open-source plugin (58k+ GitHub stars) adding Brainstorm → Plan → Execute methodology with Socratic questioning, automatic worktree isolation, mandatory TDD, and two-stage review. Best for medium-to-large, multi-file features. Caveats: sequential execution, mandatory TDD, brainstorming can consume ~43% of context.
- **RALPH Loop:** A bash script that runs Claude in a loop until a completion condition is met. Takes a `prd.json` with user stories, acceptance criteria, and status tracking. Always set `--max-iterations` to cap spending.
- **GSD Framework:** Creates a `.planning/` folder with roadmap, state, per-phase plans, and UAT files. Better when scope is ambiguous and Claude needs to help with planning.
