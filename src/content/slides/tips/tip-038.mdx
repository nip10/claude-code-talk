_Source section: MCP Servers_
- **What:** Ask Claude to find and install MCPs, or use `/mcp` to inspect and manage them.
- **Why it matters:** Removes the need to manually manage config files and gives you visibility into what tools Claude has access to.
- **How:** Say "Find me a good Figma MCP and install it" for AI-assisted setup. Run `/mcp` to see installed MCPs, or `/mcp add [service]` to install new ones directly.
