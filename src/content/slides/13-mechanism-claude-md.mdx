<TipSlide
  title="CLAUDE.md: Persistent Project Context"
  description="Use CLAUDE.md for stable defaults and constraints that should apply across tasks."
  tags={["Context", "Foundations"]}
>

Use when: you want consistent behavior every time Claude starts work in this repo.

Avoid when: the instruction is only useful for one specific workflow.

```markdown
# CLAUDE.md

## Working Agreement
- Keep changes small and focused.
- Run `npm run lint` before finalizing.
- Prefer editing existing files over adding new abstractions.

## Communication
- Explain trade-offs briefly before big changes.
```
