<TipSlide
  title="Front-end Focus: Verification First"
  description="For UI work, code is not done until the interface is verified."
  tags={["Front-end", "Quality", "Verification"]}
>

- Visual correctness: spacing, hierarchy, states, and regressions.
- Interaction correctness: clicks, forms, keyboard flow, and edge cases.
- Content correctness: labels, empty states, truncation, and copy quality.
- Responsive correctness: breakpoints, overflow, and layout behavior.
- Runtime correctness: console errors, network failures, and hydration issues.

</TipSlide>
